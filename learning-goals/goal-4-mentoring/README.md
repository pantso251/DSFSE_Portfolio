## üéì Mentoring Approach

### Guiding Principles
1. **Learning by doing** - Guide through hands-on tasks rather than lecturing
2. **Clear documentation** - Write code with extensive comments
3. **Reproducible workflows** - Create step-by-step notebooks they could follow
4. **Patience and encouragement** - Build confidence through small successes
5. **Ask questions** - Encourage understanding, not just copying code
