# DSFSE_Portfolio
Personal Learning Trajectory for Data Science For Smart Environments Course

This repository documents my personal learning journey through the Data Science For Smart Environments (DSFSE) course. It includes assignments, projects, notes, and reflections throughout the course.

## Repository Structure

- **`weeks/`** - Weekly materials, notes, and exercises organized by week
- **`assignments/`** - Course assignments and solutions
- **`projects/`** - Course projects and implementations
- **`notes/`** - Personal study notes and resources
- **`LEARNING_LOG.md`** - Weekly learning log and reflections

## About This Course

Data Science For Smart Environments focuses on applying data science techniques to smart environment applications, including IoT, sensor data analysis, and intelligent systems.
